# Hangman_Game-c
Simple hangman Game with C

  TCP-IP SERVER CLIENT game.
  
  - Implements three basic advanced programming objects :  
                                                           
                                                           Message Queues
                                                           Shared Memory
                                                           Signals
    Complile and run
    
        gcc -o h hangman.c
        gcc -o c clinet.c
     
    Pass the dictionary for the server while execute it
    
        ./h dictionary.txt
